# Simple Menu

## Sobre o Projeto

Este é um projeto simples de menu em console, desenvolvido com o objetivo de praticar lógica de programação. O sistema permite:

- Adicionar clientes

- Adicionar produtos

- Listar todos os clientes cadastrados

- Listar todos os produtos cadastrados

Ideal para iniciantes que desejam entender melhor como funcionam estruturas de repetição, condicionais e armazenamento de dados em memória.
