
# Simple Menu

## Sobre o Projeto

Este é um projeto simples de menu em console, desenvolvido com o objetivo de praticar lógica de programação. O sistema permite:

- Adicionar clientes

- Adicionar produtos

- Listar todos os clientes cadastrados

- Listar todos os produtos cadastrados

## Tecnologias Utilizadas

- Linguagem de Programação: **C**
- Compilador compatível: **GCC**
